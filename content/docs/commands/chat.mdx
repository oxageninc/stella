Start an interactive REPL session. This is the **default** command: running `stella` with no subcommand is exactly the same as running `stella chat`.

## Synopsis

```bash
stella chat [global flags]

# Equivalent — chat is the default
stella [global flags]
```

## What it does

`stella chat` opens a conversational loop. You type a message, the agent runs its step loop — proposing tool calls, executing them, and feeding results back until the turn is done — and then hands the prompt back to you for the next message. Conversation history is retained across turns within the session.

Inside the REPL you also have a set of **slash commands** for inspecting state and controlling the session without leaving it.

<Callout type="info">
All [global flags](/docs/commands) apply. For example, `stella chat --model openai/gpt-5.5` pins the worker model for the whole session.
</Callout>

## Flags

`stella chat` takes only the [global flags](/docs/commands). Commonly used here:

| Flag | Why it matters here |
| --- | --- |
| `--model <provider/model_id>` | Pin the worker model for the session. |
| `--budget <usd>` | Apply a hard spend cap across the entire session. |
| `--base-url <url>` | Required with `--model local/<model>`; an optional proxy override otherwise. |

## REPL slash commands

Type these at the REPL prompt inside a chat session:

| Command | Description |
| --- | --- |
| `/help` | Show help. |
| `/models` | List providers and models (same output as `stella models`). |
| `/config` | Show the current configuration. |
| `/clear` | Clear the conversation history. |
| `/files` | List the files changed this session. |
| `/goal` | Switch into goal mode from chat. |
| `/exit` | Leave Stella. |
| `/quit` | Leave Stella (alias of `/exit`; the bare word `exit` also works). |

## Examples

Start a session with the default (auto-detected) model:

```bash
stella chat
```

Start a session by simply running the binary (chat is the default):

```bash
stella
```

Pin a model and cap session spend:

```bash
stella chat --model zai/glm-5.2 --budget 5.00
```
